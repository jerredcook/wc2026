The Refresh latest scores button pulls current scores — including matches in progress (red LIVE badge + live minute) — from a public scoreboard when the page is served over http(s), or via Claude when open in Claude's view. A file opened straight from disk (file://) is blocked by the browser from making web requests, so it keeps the saved scores; the status line always tells you which it is.
